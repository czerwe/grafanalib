Welcome to grafanalib's documentation!
======================================

Primary grafanalib provides an api implementation for basic administration of an Grafana instance. As an additional help it's providing helper classes to create Datasources and dashboards with or without row's and panels.

Currently supported Datasources

* Graphite
* InfluxDB

Currently supported panels

* Graph
* Text

For everything else it's possible to use the api provided by the grafanalib inclusion.
